# Результаты обучения YOLOv8-seg

**Модель:** `yolov8n-seg.pt`
**Эпох:** 2 (демонстрационный прогон)
**Устройство:** CUDA:0 (NVIDIA GeForce RTX 4060)

## Метрики (по окончании 2 эпох)

Модель обучилась на 644 изображениях (1019 объектов) из 20 классов.

- **Mask mAP50:** 0.524
- **Mask mAP50-95:** 0.478
- **Box mAP50:** 0.523
- **Box mAP50-95:** 0.464

### Выводы
1. **Скорость обучения:** Использование CUDA радикально ускорило процесс. 2 эпохи были пройдены буквально за пару минут (0.025 hours). Время инференса на батч составляет всего ~1.8ms per image, что отлично подходит для мобильного API.
2. **Точность:** Для всего 2 эпох метрика `Mask mAP50` = 0.52 — это отличный старт. Нейросеть уже уверенно выделяет контуры крупных продуктов (cauliflower: 0.90, guava: 0.84, potato: 0.77).
3. **Проблемные классы:** Классы типа `garlic-1`, `onion-1`, `tomato-1` показывают околонулевую полноту. Это ожидаемо для малого количества эпох и малого количества примеров (например, для `cauliflower-1` всего 3 примера).
4. **Готовность:** Модель сохранена в `runs/segment/food_segmentation_test/weights/best.pt` и готова к интеграции в `inference.py` для тестирования логики расчета объема.
